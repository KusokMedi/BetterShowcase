# BetterShowcase

> A fork of [xdBot](https://github.com/ZiLko/xdBot) with improvements and new features.

**BetterShowcase** is a Geometry Dash mod for showcases and macro recording, built on the Geode mod loader.

## Features

* Basic ClickBot
* Practice Fixes
* Seed Modifier
* Noclip
* Show Trajectory
* Layout Mode
* Speedhack
* Frame Stepper
* Safe Mode
* Renderer
* Instant Respawn
* No Respawn Flash
* No Death Effect
* Macro saving and loading system
* Macro Auto Saving

## Improvements over xdBot

* ✅ Linux support
* ✅ FFmpeg API mod integration (no need for ffmpeg.exe)
* ✅ Choose between Mod or Custom FFmpeg mode
* ✅ Updated branding and links

## Installation

1. Install [Geode](https://geode-sdk.org/) mod loader
2. Download BetterShowcase from the in-game mod browser
3. (Optional) Install [FFmpeg API](https://geode-sdk.org/mods/eclipse.ffmpeg-api) for easier rendering

## How to Use

1. Open the menu using the Open Menu Keybind or the button in the Pause Menu
2. Click the 'Record' toggle in the menu to start recording
3. Finish a practice run of the level
4. Play the macro by clicking the 'Play' toggle in the menu and going into the level
